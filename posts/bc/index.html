Beauty Club — Сайт студии красоты

September 20, 2025 · 3 min · wizoomm

Демо#

Сайт: https://se.ifmo.ru/~s368382/bc/index.html
Репозиторий: https://github.com/wizoomm/bc.git

Описание проекта#

Учебный проект — веб-сайт для студии красоты “Beauty Club” — студии маникюра, педикюра и лазерной эпиляции, расположенной в Санкт-Петербурге. Сайт предоставляет полную информацию о студии, услугах, специалистах и позволяет клиентам оставлять заявки на консультацию.

Основные возможности#

  • Адаптивный дизайн — сайт корректно отображается на всех устройствах (десктоп, планшет, мобильные)
  • Интерактивная форма записи — валидация данных на клиенте и сервере
  • Интеграция с базой данных — сохранение заявок в PostgreSQL
  • Многостраничная структура — удобная навигация по разделам сайта
  • Галерея работ — демонстрация портфолио студии
  • Информация о специалистах — профили мастеров с опытом работы

Технологический стек#

Frontend#

  • HTML5 — семантическая разметка
  • CSS3 — стилизация с использованием кастомных шрифтов (Helvetica Neue)
  • JavaScript (Vanilla) — интерактивность без фреймворков
    • Валидация форм в реальном времени
    • Маски ввода для телефона
    • Управление состояниями формы (форма/успех)
    • Адаптивное бургер-меню

Backend#

  • PHP — обработка форм и работа с базой данных
  • PostgreSQL — хранение заявок на консультацию

Функциональность#

Форма записи на консультацию#

Форма включает:

  • Валидацию имени: только буквы, минимум 2 символа
  • Валидацию телефона: 11 цифр, начинается с 7 или 8
  • Маски ввода: автоматическая фильтрация недопустимых символов
  • Обработку состояний: форма → отправка → успех/ошибка
  • Сохранение в БД: заявки сохраняются в таблицу consultations

Адаптивная навигация#

  • Десктоп: горизонтальное меню в хедере
  • Мобильные устройства: бургер-меню с анимацией
  • Breadcrumbs на внутренних страницах

Страницы сайта#

  1. Главная — баннер, информация о студии, команда, услуги, галерея, отзывы
  2. О нас — подробная информация о студии, основателе, преимуществах
  3. Услуги — полный прайс-лист (маникюр, педикюр, лазерная эпиляция)
  4. Запись — форма для записи на консультацию
  5. Акции — текущие предложения и скидки
  6. Рекомендации — советы по уходу
  7. Вопросы — FAQ
  8. Галерея — портфолио работ
  9. Отзывы — отзывы клиентов
  10. Специалисты — профили мастеров

Особенности реализации#

Валидация форм#

  • Клиентская валидация: проверка в реальном времени при вводе
  • Серверная валидация: дополнительная проверка на стороне PHP
  • UX улучшения: кнопка отправки активируется только при валидных данных
  • Обработка ошибок: отображение сообщений об ошибках через URL-параметры

Работа с базой данных#

// Структура таблицы consultations
CREATE TABLE consultations (
    id SERIAL PRIMARY KEY,
    full_name VARCHAR(255) NOT NULL,
    phone VARCHAR(20) NOT NULL,
    created_at TIMESTAMP DEFAULT NOW()
);
